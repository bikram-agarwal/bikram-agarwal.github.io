FilePipe

Android

Turn messy Android storage into a rule-driven library. Create rules for downloads, screenshots, documents, videos, music, or any folder you choose. Filter by extension, name patterns, size, age, and subfolders, then run manually or on a schedule.

Why It Exists

Different apps save files in separate app folders, and browser downloads dump everything — zips, PDFs, APKs — into one cluttered Download folder. I built FilePipe to automatically gather scattered media into one place and sort downloads into organized, type-specific folders.

Preview Before Changes

See which files will be affected before a single byte is moved. Change your mind? Cancel any run mid-batch, instantly.

History and Undo

Every run is logged with file-level results — filter by success or failure, group by date, rule, or status, and undo supported moves directly from the log.

Scheduled Cleanup

Run rules hourly, daily, or weekly. Scheduled jobs post summary notifications with a one-tap undo button, and manual runs keep working even if you switch apps.

Made for Power Users

Templates, swipe shortcuts, batch actions, exports, custom icons, and detailed filters keep advanced workflows fast.
