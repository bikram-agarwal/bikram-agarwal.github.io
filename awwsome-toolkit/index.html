AWWsome Toolkit

Android Windows Web

A collection of small, platform-specific tools and guides for Android, Windows, and the web — built for tasks that benefit from repeatable automation and written to be understood before they're run.

Why It Exists

I built these tools to keep my own system tidy. Windows kept rearranging my Start Menu every time I installed or updated something, and the registry tweaks I found online lived in scattered notes until I scripted them for myself. Eventually I realized I was building a small library of fixes for everyday annoyances — and that others might need them too.

Android Utilities

A PowerShell script that automates Google Pixel boot image patching for gaining root access, plus documentation for extracting AI models from Pixel device factory images.

Windows Automation

Enforce Start Menu layouts, repair moved-drive shortcuts, inventory installed apps, reset Xbox/Game Pass, toggle hidden Windows features, bootstrap Android tooling, and audit or clean driver/system caches.

Web Workflows

A Google Apps Script that syncs calendar events into a Google Tasks Backlog — bidirectionally: tasks complete when events are marked done, and vice versa. Includes a themed browser dashboard.

Built for Review

Every tool is meant to be understood before it runs, with safety notes where a step is hard to undo.
