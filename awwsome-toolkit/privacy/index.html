Privacy Policy

Last updated: May 9, 2026 | for Project: AWWsome Toolkit

Summary

AWWsome Toolkit is a collection of Android, Windows, and web utilities. Some tools run locally on your own device, some are meant to be copied into your own Google Apps Script project, and the release-feed utility runs in this repository's GitHub Actions workflow. The developer does not collect analytics, run ads, or send your data to any server not described in this policy.

Local scripts and guides

Android and Windows scripts in this toolkit run in the environment where you execute them. Depending on the script, they may read local files, inspect system configuration, create backups, download required tools, modify folders, or clean up system data. Review each script before running it. Data handled by these scripts is not sent to the developer's servers.

Google Apps Script web app

The Calendar to Task Sync tool is designed to be copied into a Google Apps Script project that you own. If you deploy it, Google runs the script under your Google account and it accesses the Google Calendar and Google Tasks data you authorize it to use.

The web interface can display calendar event titles, task titles, dates, task notes, and sync results in your browser. It stores theme and display preferences in your browser's local storage. That data is handled by your browser and by Google Apps Script, not by a server run by the developer.

Execution logs for the Google Apps Script tool are handled by Google Apps Script in your Google account. Google's own privacy policy applies to data Google holds.

Release-feed utility

The release-feed utility runs in this repository's GitHub Actions workflow. It fetches public GitHub star lists, public starred repositories, and public release metadata for the configured GitHub account, then publishes generated Atom feeds and a static index page through GitHub Pages.

The workflow uses GitHub's built-in token for GitHub API rate limits. It does not use your personal credentials unless you fork or modify the workflow yourself. GitHub Actions and GitHub Pages are operated by GitHub, and GitHub's own privacy policy applies to build logs, API requests, and page hosting.

Third-party services

Some tools may call third-party services, package registries, platform APIs, download hosts, Google services, or GitHub services depending on how you use them. Those services are governed by their own privacy policies.

No analytics or tracking

The project does not include analytics, advertising, crash-reporting, or telemetry SDKs. The developer does not receive information about how you use the project, except for information you intentionally provide through GitHub issues, pull requests, or other direct communication.

Children

This project is not directed at children under 13, and the developer does not operate any service that collects personal information from children.

Changes

This policy may be updated from time to time. The latest version will be published on this page.

Contact
