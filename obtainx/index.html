ObtainX

Android

ObtainX builds on everything Obtainium does well — same sources, same trust model, same spirit — with a reworked interface and features aimed at making everyday use smoother.

App Update Management

Track apps from GitHub, F-Droid, APKMirror, and more. ObtainX surfaces six distinct update states instead of a binary yes/no, shows download size before you commit, and supports custom installers. Already on Obtainium? Import your full app list in seconds and continue exactly where you left off.

GitHub Privacy Terms

Bulk Import

Select apps already on your phone and ObtainX finds their sources automatically — no URL hunting, one by one. APKMirror supported too, with the Update button taking you to the exact release page.

Smarter Updates

Six distinct states instead of a binary pair. See exact download size upfront, skip a specific version without losing tracking, and route installs through any trusted installer you choose.

Cleaner App Lists

Folders with auto-assignment rules (by name, author, or source), configurable left/right swipe actions, filter chips, and on-demand mode keep large app lists navigable and clutter-free.

Modern Android UI

Per-app icon color theming, hero icon transitions, inline editing, source favicon badges on every row, and settings grouped into scannable cards.
